The Escape Theorie

Dashboard Setup

Einmalige Anmeldung bei Metricool — danach läuft das Dashboard
vollautomatisch, täglich, für immer.

1
Browser öffnet sich automatisch
Klick auf den Button unten. Ein Metricool-Fenster öffnet sich auf deinem Bildschirm.
2
Einloggen wie gewohnt
Verwende deinen Beziehungscode oder eine andere Login-Methode — genauso wie immer.
3
Fertig — Session wird automatisch gespeichert
Sobald du im Dashboard bist, erkennt das System den Login und speichert alles dauerhaft.

Deine Session wird ausschließlich lokal auf diesem Server gespeichert
und nie an Dritte übertragen.
